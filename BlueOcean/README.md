# 청소년의 안전한 모임 설계 플랫폼

이 프로젝트는 React 프론트엔드와 python crwaling 코드/ gpt_api를 이용한 청소년을 위한 플랫폼입니다.
백엔드 서버는 구성되어 있지 않고 crwaing과 api를 활용한 summary코드만 들어 있습니다.
---

## 📁 프로젝트 구조

babybaby/
├── frontend/ # React 기반 사용자 인터페이스
│ └── src/
│ ├── App.js
│ ├── App.css
├── backend/ # crwaling 코드와 데이터 summary 코드
│ ├── crwaling.ipynb
│ ├── app.py


---

## 🚀 실행 방법

### 1. 크롤링 코드 실행
crwaling.ipynb
python app.py

### 2. 프론트엔드 실행 (React)
cd frontend
npm install
npm start
